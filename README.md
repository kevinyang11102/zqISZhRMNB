# 学生信息管理系统 python19

## 项目概述

学生信息管理系统是一款基于 Python 语言和 MySQL 数据库开发的应用，主要用于高校或教育机构管理学生相关信息。

## 技术栈

- 后端：Python
- 数据库：MySQL
- 前端：HTML, CSS, JavaScript（可选）

## 功能模块

- 学生信息管理
  - 添加学生信息
  - 修改学生信息
  - 删除学生信息
  - 查询学生信息
- 成绩管理
  - 录入成绩
  - 修改成绩
  - 删除成绩
  - 成绩查询
- 报告生成
  - 学生信息报告
  - 成绩报告

## 系统角色

- 管理员：负责系统的维护和所有学生信息的管理
- 教师：负责录入、修改和查询学生成绩
- 学生：可查询个人信息和成绩

## 运行环境

- 操作系统：Windows、Linux 或 macOS
- Python 环境：Python 3.x
- 数据库：MySQL 5.x 或更高版本

## 部署步骤

1. 安装 Python 3.x
2. 安装 MySQL 5.x 或更高版本
3. 创建数据库和对应的数据表
4. 克隆或下载项目源码
5. 安装项目依赖库
6. 运行系统
